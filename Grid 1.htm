  • Description: Этот модуль CSS определяет двумерную систему компоновки на основе сетки, оптимизированную для дизайна пользовательского интерфейса. В модели компоновки сетки дочерние элементы контейнера сетки могут быть расположены в произвольных ячейках в предопределенной сетке компоновки гибкого или фиксированного размера. CSS - это язык для описания рендеринга структурированных документов (таких как HTML и XML) на экране, на бумаге и т.д.
